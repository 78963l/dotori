# dotori

![travisCI](https://secure.travis-ci.org/rd101/dotori.png)

Dotori is web based asset library tool.

## 목 표
- VFX, 애니메이션, 게임에 사용되는 모든 타입의 에셋을 등록,관리
- 사용라이브러리: OpenColorIO, OpenImageIO, FFmpeg

## DB 설치, 실행
mongoDB

## Dotori 실행

```bash
$ dotori -http :80
```
